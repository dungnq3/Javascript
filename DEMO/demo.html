Yêu cầu: Nghiên cứu các hàm truy xuất các phẩn tử trong html theo bộ chọn selector mong muốn bằng js và jquery

Nội dụng nghiên cứu:

+ Selector là gì? -> Dựa vào phân cấp theo tag, id, class để truy xuất phần tử html

+ DOM là gì? -> Dùng để xử lý các phần tử trong html thông qua đối tượng document

+ DOM Element -> Truy xuất đến thẻ html thông qua thuộc tính id, class, tag name, cú pháp CSS selector

- Ví dụ: document.getElementById('id'), document.getElementsByTagName('tag name'), document.getElementsByClassName('class'),document.querySelectorAll("selector.css");

+ DOM HTML -> Xử lý nội dung và thuộc tính của thẻ

- Ví dụ: .innerHTML: nội dung, .attributeName: thuộc tính như name, value, type,...

+ DOM CSS -> xử lý css qua đối tượng style

- Ví dụ: .style.cssName

+ DOM Node -> là đối tượng trả về từ DOM Element

- Ví dụ: tạo node mới document.createElement("p"); p.appendChild(): Thêm vào vị trí cuối cùng, insertBefore(insert,child), thêm vào trước child, removeChild(), replaceChild();

+ Bộ trọn selector trong jquery Bộ chọn select

+ Hàm(selectors) trong jquery

Tổng kết: Biết cách chọn selector(xác định các thẻ) -> lấy được đối tượng(các thẻ html), xác định được parent và child -> Xử lý sự kiện
